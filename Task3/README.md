# Calculator Web Application

## Project Description
A colorful calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations and includes calculation history.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear Button
- Delete Button
- Calculation History
- Responsive Design
- Colorful User Interface
