# Travel Landing Page

## Project Description
A modern and responsive travel landing page developed using HTML and CSS. The page includes destination highlights, about section, places to visit, contact information, and attractive UI.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Responsive Design
- Hero Section
- About Us
- Popular Destinations
- Places to Visit
- Contact Section
- Attractive Layout
- Smooth Navigation
